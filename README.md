# Calculadora de Lei de Ohm (U = R × I) – Kotlin Android

Este é um aplicativo Android desenvolvido em Kotlin que permite calcular qualquer uma das variáveis da **Lei de Ohm** (Tensão, Corrente ou Resistência) a partir das outras duas. O app é simples, educativo e ideal para quem está aprendendo Kotlin e desenvolvimento Android.

---

## 📱 Funcionalidades

- Campo para digitar **Tensão (U)**
- Campo para digitar **Resistência (R)**
- Campo para digitar **Corrente (I)**
- Botão **Calcular** que descobre automaticamente qual variável está faltando
- Validação de entradas:
  - É necessário preencher **pelo menos 2 campos**
  - Valores negativos ou inválidos não são aceitos
- Resultado exibido na tela com 2 casas decimais
- Botão estilizado em vermelho com texto branco e cantos arredondados

---

## 🛠 Tecnologias utilizadas

- Kotlin  
- Android Studio  
- ConstraintLayout  
- Material Components (para botão estilizado)
